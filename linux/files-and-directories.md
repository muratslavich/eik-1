# Files and Directories

* Delete

```
# single file
$ rm filename

# several files
$ rm file1 file2 file3
$ rm *.txt

# ask to confirm each before deleting
$ rm -i *.txt

# remove write protected file
$ rm -f file

# empty directory
$ rm -d dirname
$ rmdir dirname

# non empty directory
$ rm -r dirname

# without being prompted files
$ rm -rf dirname
```
